# WINDOWS-OEM-PRODUCT-KEY-INJECTOR
Ini adalah tool untuk menginject lisensi product key windows ke motherboard agar product key di inject ke BIOS / Motherboard sehingga product key tidak menghilang ketika install ulang windows dan langsung teraktivasi.
Cara kerja tool ini: Tool ini menggunakan basic software dari AFUWin untuk menyuntikan lisensi product key ke motherboard
Cara menggunakan tool ini :
- Buka file KEY.XLSX menggunakan Excel atau WPS Office
- Masukkan Product key yang ingin di inject dan Serial number product
- Close Excel atau WPS Office nya
- Install git di komputer kalian
- ketik perintah berikut :
  # git clone https://github.com/fara2001/WINDOWS-OEM-PRODUCT-KEY-INJECTOR
  # cd WINDOWS-OEM-PRODUCT-KEY-INJECTOR
  # offline.bat
* jika menggunakan powershell bisa ketik .\offline.bat
- Cek product key sudah di inject dengan aplikasi produkey.exe
- Cek aktivasi di windows settings
